## 二、四大研究方向详述

### 方向一：供应链断裂成因与影响（Q1相关）

**核心研究问题**：贸易政策冲击、自然灾害、公共卫生事件等外部冲击如何沿供应链纵向传导？对企业经营绩效和可持续性的影响程度与异质性模式如何？

**子方向分解：**

| 子方向 | 代表论文 | 核心发现 |
|-------|---------|---------|
| **贸易政策冲击与断链** | 丁浩员等（2024） | 美国对华加征关税显著提高中美供应链断裂概率；产品替代性、契约成本和合作基础是韧性关键因素 |
| **自然灾害沿供应链传播** | 魏龙等（2024, 中国工业经济） | 自然灾害冲击沿供应链纵向传播，增加跨区域采购边际成本，破坏商业信用资金链 |
| **公共卫生事件双重冲击** | 刘瑶等；曾嶒等（2025） | 新冠疫情造成供给侧中断+需求侧疲软双重叠加；国有企业在供应链中发挥"稳定器"效应 |
| **国家安全泛化与数据供应链** | 张明等 | 国家安全概念的泛化正引发全球数据供应链重构，数据主权成为新的断裂驱动因素 |

**研究空白**：多数研究聚焦于单一冲击类型（关税/灾害/疫情），缺乏对**多类型冲击叠加效应**的综合检验；断链的**微观传导速度与时滞**研究不足。

---

### 方向二：供应链重构机制与路径（Q2相关）

**核心研究问题**：企业通过哪些机制和路径重构断裂的供应链？（多元化、近岸转移、本土替代？）重构的驱动因素和约束条件是什么？

**子方向分解：**

| 子方向 | 代表论文 | 核心发现 |
|-------|---------|---------|
| **供应链多元化策略** | 沈国兵等（2025）、魏龙等（2024） | 数字化通过提升信息搜寻、创新和议价能力促进供应链多元化；多元化是提升韧性的有效选择 |
| **近岸/本土/第三国转移** | 丁浩员等（2024） | 断链后中美关系难以短期恢复；企业倾向于将供应链转移至近岸（东南亚）、本土及其他欧美发达国家 |
| **纵向分离与聚焦** | 姚雨秀等（2025, 数量经济技术经济研究） | 数字化通过增强核心能力和降低交易成本推动纵向分离，优化供应链纵向业务配置 |
| **可持续供应链治理** | 李婧婧等（管理评论 2021） | 可持续治理从核心企业→供应链成员→供应链生态系统三阶段演进，资源编排驱动治理扩展 |
| **全球中高端制造重构** | 杨长春等 | 全球中高端制造业供应链呈现"区域化+本土化"重构趋势 |

**研究空白**：
1. 重构路径的**动态演化过程**揭示不足（多数为静态比较）
2. 重构后供应链的**绩效评估**（效率vs韧性的权衡）缺乏系统证据
3. 中国客户企业作为**采购方**重构上游供应链的专属分析较少

### 方向三：数字化转型的供应链溢出效应（跨领域赋能）

> 本知识库论文数量最多的方向（约50%的论文涉及），是当前研究热点。

**核心研究问题**：企业数字化转型沿供应链上下游产生何种溢出效应？其传导机制和异质性特征如何？

**子方向分解：**

| 子方向 | 代表论文 | 核心发现 |
|-------|---------|---------|
| **生产力溢出（后向溢出）** | 陶锋等（2023, 中国工业经济）；罗双成等 | 下游企业数字化转型→通过供需匹配优化→牵引上游企业TFP增长（"后向溢出效应"） |
| **绿色溢出效应** | 蒋真儿等（2026, 财经论丛） | 数字化转型降低上游交易成本+提升内部控制→促进供应商绿色全要素生产率增长 |
| **碳减排双向溢出** | 李治国等（2025） | 共同供应链网络中数字化转型产生碳排放的双向溢出和双重红利（↓总量↓强度） |
| **跨国投资联动效应** | 梁贺等（2025） | 数字化转型通过供应链关联产生对外直接投资的"抱团出海"联动效应 |
| **GVC位置攀升效应** | 孙林等（2025） | 数字化转型+数字基础设施→TFP与管理效率提升→企业向GVC上游攀升 |
| **出口韧性提升** | 赵亚娟等（2025） | 供应商数字化转型→生产网络稳定性+供应链话语权→下游企业出口韧性增强 |
| **供应链风险** | 梁贺等（2024） | 数字化转型降低供应链风险，但需警惕"数字化双重效应"——溢出效应与风险传染并存 |
| **客户关系稳定性** | 丁怡帆等 | 数字化→降低信息摩擦→提升客户关系稳定度 |
| **纵向分离** | 姚雨秀等（2025） | 数字化转型→增强核心能力+降低交易成本→推动后向分离 |

**研究空白**：
1. 数字化溢出效应的**边界条件**研究不足（何时溢出失效？何时产生负外部性？）
2. 溢出效应的**动态演化**（数字化程度的非线性阈值效应）缺乏探讨
3. **行业异质性**分析多但**机制异质性**解析少

### 方向四：产业政策与制度环境对供应链的影响（Q3相关）

**核心研究问题**：产业政策（开发区、链长制、补贴等）如何影响供应链重构？政策传导的微观机制是什么？

| 政策类型 | 代表论文 | 核心发现 |
|---------|---------|---------|
| **开发区政策** | 卞泽阳等（2024） | 开发区主导产业的正向冲击→通过供应链关系→缓解上下游企业融资约束（上游12.5%,下游8.5%） |
| **国有企业角色** | 曾嶒等（2025） | 国有企业通过保障商品流转效率+提供商业信用→扶持供应链上民营企业→稳定其经营现金流 |
| **数字化转型政策** | 陶锋等（2023） | 建议以下游企业数字化为突破口，推动纵向协同机制建设，注意分类施策 |
| **产业链安全保障** | 黄庆华等；蒋建军等；刘家国等 | 从"双循环"视角设计韧性体系（内循环动态迭代+外循环四位一体+内外联通一体两翼） |

**研究空白**：
1. 政策效果的**因果识别**仍偏弱（多数为政策建议型，缺少严格反事实分析）
2. 政策传导的**供应链乘数效应**缺乏量化
3. 产业政策与市场机制之间的**替代/互补关系**未有清晰界定

## 三、研究方法前沿趋势

从本知识库的论文可以看出以下前沿方向：

1. **从"企业内"到"企业间"**：研究单位从单个企业扩展到供应链上下游配对（dyad）和供应链网络（network）
2. **从"相关关系"到"因果识别"**：DID、IV等准实验方法主导，但需要更多自然实验/政策冲击设计
3. **从"单一绩效"到"多维韧性"**：绩效指标从传统的效率/利润转向韧性（恢复速度、适应能力）
4. **从"定性/定量二分"到"混合方法"**：大数据+深度案例的混合设计开始出现
5. **供应链关系数据的精细化**：从年报前五名客户/供应商向FactSet等全覆盖数据库拓展

## 四、研究空白与未来方向

### 4.1 直接相关于本知识库核心命题（中国客户→全球供应商重构）

| 研究空白 | 紧迫性 | 可能的切入路径 |
|---------|--------|--------------|
| 中国客户企业作为**采购方**的供应链重构专属研究 | ★★★★★ | 聚焦"买方视角"，区分"进口替代"与"供应源切换"两种重构模式 |
| **断链→重构的全过程追踪** | ★★★★☆ | 利用FactSet/海关数据的动态DID，追踪同一供应链关系断链前后的演化 |
| **政策组合的供应链效应** | ★★★★☆ | 开发区+产业基金+贸易政策的多重冲击叠加分析 |
| **效率-韧性权衡** | ★★★☆☆ | 构建双重绩效指标，识别最优多元化程度 |

### 4.2 方法论空白

| 空白 | 说明 |
|------|------|
| **供应链网络的动态建模** | 现有多数为静态关联分析，缺乏对网络拓扑演化的动态建模 |
| **跨境供应链关系的微观数据** | FactSet Revere覆盖有限，多数研究依赖年报间接推断 |
| **调节效应的理论化** | 异质性分析多但缺乏对"为何异质"的理论解释 |
