# 案例C：DeepSeek/长鑫存储等暂缓列入实体清单

> 2026年6月，路透社报道美国政府暂缓将DeepSeek、长鑫存储（CXMT）及其他100多家被认定存在国家安全风险的中国企业列入实体清单，创下实体清单发布"最长停滞期"。

---

## 一、背景

### 1.1 制裁冻结期的出现

自2024年10月以来，美国BIS未在其"实体清单"中增加任何新成员，这创下了十多年来新版名单发布的"**最长停滞期**"。^[来源：断供规模数据.md，第5.1节]

商务部副部长凯斯勒自2025年底以来倾向避免进一步加剧美中关系紧张。

### 1.2 积压的待制裁实体

截至2026年6月，至少**75家**涉及先进半导体制造、设备生产及AI模型开发的中国实体已通过跨部门委员会审议，但商务部至今未予公布。^[来源：断供规模数据.md，第5.2节]

---

## 二、断裂事件（潜在断裂）

### 2.1 暂缓列入的实体范围

涉及**100多家**中国实体，覆盖领域：^[来源：断供规模数据.md，第5.2节]

| 领域      | 代表性企业      | 技术类别     |
| ------- | ---------- | -------- |
| AI大模型   | DeepSeek   | 大型语言模型训练 |
| 存储芯片    | 长鑫存储（CXMT） | DRAM制造   |
| 半导体设备   | 多家未具名企业    | 晶圆制造设备   |
| AI初创公司  | 多家未具名企业    | AI算法与应用  |
| 无人机/机器人 | 多家未具名企业    | 智能无人系统   |

### 2.2 暂缓原因分析

| 可能因素 | 说明 |
|---------|------|
| **外交缓和** | 中美高层对话增加，避免制裁加剧关系紧张 |
| **内部分歧** | 跨部门委员会对制裁范围和程度存在分歧 |
| **反制考量** | 中国《产业链供应链安全法规定稿》（第834号令）具备系统性反制能力 |
| **企业合规** | 部分企业主动配合美国出口管制合规审查 |

^[来源：断供规模数据.md，第5.2节]

---

## 三、影响

### 3.1 冻结期≠政策转向

断供规模数据.md明确指出：^[来源：断供规模数据.md，第5.2节]

> **冻结期≠政策转向，而是权衡利弊后的临时调整**

- 积压的75+实体一旦解冻集中释放，冲击更大
- 被点名企业的法律风险未解除，只是延后
- 美国防部1260H清单（中国军事企业清单）同期从扩容至**188家**，新增**64家中国企业**——从另一渠道持续施压

### 3.2 对企业的影响

| 维度 | 影响 |
|------|------|
| **法律风险** | ⚠️ 未解除，只是延后执行 |
| **供应链规划** | 不确定性增加——企业难以制定中长期备选方案 |
| **融资** | 1260H清单独立运作，对涉军企业的融资和供应链产生独立影响 |
| **市场信心** | 被点名企业的海外客户可能主动避险 |

### 3.3 2024-2026年制裁特征变化

| 特征 | 2024年之前 | 2024-2026年 |
|------|-----------|-------------|
| 实体清单新增节奏 | 高频次、大规模 | **冻结期**（2024.10至今无新增） |
| 制裁重心 | 实体清单为主 | **多清单协同**（1260H+NS-CMIC+实体清单） |
| 制裁领域 | 半导体/AI/量子 | 扩展至：AI大模型、无人机、机器人、新能源汽车 |
| 反制博弈 | 中国有限反制 | 中国**产业链供应链安全法规定稿**（第834号令） |
| 政策不确定性 | 相对可预期 | **冻结期带来不确定性**——暂停≠取消 |

^[来源：断供规模数据.md，第5.5节]

---

## 四、重构路径（预防性）

由于制裁尚未正式实施，受影响企业的重构处于**预防性储备阶段**：

1. **多元供应商储备**：可能提前接触非美供应商
2. **自研能力建设**：利用冻结期加速关键技术的自主攻关
3. **法律合规准备**：咨询出口管制律师，建立内部合规体系
4. **产能转移**：如长鑫存储可能加速国产设备替代验证

---

## 五、结果

**当前状态（2026年7月）**：

- 实体清单冻结期已持续约**21个月**（2024年10月-2026年7月）
- 75+实体审议通过但未公布，构成"悬而未决"的风险缓冲
- 美国对华管制从"实体清单单点打击"转为"多清单协同施压"
