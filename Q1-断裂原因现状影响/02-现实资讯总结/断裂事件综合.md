# 断裂事件综合

> 现实资讯层面：基于《2025年世界开放报告》和断供规模数据的供应链断裂现状事实汇编。

---

## 一、断裂的直接原因

### 1.1 美国单边关税
2025年美国推出"对等关税"，以单方强制标准替代多边贸易制度框架，涵盖芬太尼关税、对全球"对等关税"、钢铝/铜/汽车额外关税及二级关税威胁。歧视性贸易政策仍显著多于自由化措施。
→ 详见：raw/02-正在参考文档/现实资讯/2025世界开放报告分析.md §2

### 1.2 制裁清单体系
美国对华制裁形成**多部门、多清单交织的八清单体系**（BIS实体清单1089个、SDN清单999个、CMC清单131家等），叠加"推定拒绝"审查政策，被列入企业几乎不可能获得出口许可证。
→ 详见：raw/02-正在参考文档/现实资讯/断供规模数据.md §1-4

### 1.3 投资审查全面趋严
2024年约**40%以上**的全球投资限制政策涉及外资审查机制；采用FDI国家安全审查制度的国家从不到20个增至46个；发达经济体对数字经济的审查尤其严格。
→ 详见：raw/02-正在参考文档/现实资讯/2025世界开放报告分析.md §2, §10

### 1.4 地缘政治风险升至首位
"基于国家的武装冲突"在《2025年全球风险报告》中由第8位跃升至第1位，地缘经济博弈推动保护主义浪潮。
→ 详见：raw/02-正在参考文档/现实资讯/2025世界开放报告分析.md §10

---

## 二、断裂的具体影响

### 2.1 对贸易量
全球货物贸易量增长预期下调近3个百分点，CEPR测算全球贸易流量收缩**5.5%-8.5%**，流经全球价值链的贸易收缩约2个百分点。
→ 详见：raw/02-正在参考文档/现实资讯/2025世界开放报告分析.md §4

### 2.2 对企业断供冲击
"推定拒绝"政策使实体清单企业无法获取EAR管辖物项；受限技术领域从传统制造逐步扩展至：**AI芯片 → 先进计算 → 量子技术 → 高超音速 → 生物技术 → 新能源材料**。
→ 详见：raw/02-正在参考文档/现实资讯/断供规模数据.md §4

### 2.3 对多边贸易体制的破坏
美国"对等关税"违反WTO最惠国待遇原则和约束税率义务，单方面突破减让表规定税率，以国内法替代国际规则。
→ 详见：raw/02-正在参考文档/现实资讯/2025世界开放报告分析.md §4

## 三、宏观背景：效率→安全

全球产供链底层逻辑从**效率优先转向效率与安全并重**，由三重冲击触发：中美贸易摩擦升级、地缘政治冲突泛化至经济与技术领域、全球疫情暴露系统脆弱性。
