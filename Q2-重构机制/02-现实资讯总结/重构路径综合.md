# 重构路径综合

> 现实资讯层面：基于新闻报道、企业公告、行业报告的供应链重构事实汇编。

## 核心论点：效率→安全并重

全球供应链重构的根本动因，在于布局逻辑从**效率优先**转向**效率与安全并重**，由三重外部冲击触发：贸易摩擦（中美关税暴露单一来源脆弱性）、地缘政治冲突（安全泛化至经济与技术领域）、新冠疫情（暴露全球化体系的系统性脆弱）。各国同步推动"自主可控"——美国《芯片与科学法案》、欧盟《关键原材料法案》、"近岸/友岸外包"政策形成合力。

## 一、宏观趋势：多元化 > 本土化

### 1.1 本土化效果不明显
2017-2023年各地区本土中间品比例变化：墨西哥 **+7.6pp**（关税规避驱动）、东亚与太平洋 +0.5pp、北美 -1.4pp、欧洲与中亚 -1.0pp、东盟 **-3.7pp**。全面本土化放弃比较优势成本红利，效果不显著。

### 1.2 多元化是主要趋势
不同区域从其他来源进口中间产品的比例**几乎全面上升**，东盟从东亚与太平洋进口 +3.4pp 最为典型。

### 1.3 驱动因素
- **效率逻辑**：全球已形成高度专业化产业集群，多源采购通过竞争性报价维持成本竞争力
- **安全逻辑**：供应商备份（2-3个可替换）+ 地理分散对冲断供风险
- **技术壁垒**：半导体光刻机等高集中度产业短期内无法本土化

> **混合模式**："有重点的本土化 + 多来源的多元化"并存。

## 二、地理格局：三大中心 + 卫星区域

```
三大中心:  北美 ──── 欧洲 ──── 东亚与太平洋
卫星区域:  墨西哥    东欧/北非  东盟
```

资本流向印证：东南亚 FDI 从 348 亿美元升至高峰 710 亿美元，2018-2024 年增量占总增量的 **57.9%**，来源从欧洲为主转向中日韩为主。供应链布局正从**全球化粗放**转向**区域化精细协同**（"短链化""圈层式"体系）。

## 三、企业层面重构策略

| 策略 | 核心逻辑 | 代表案例 |
|------|---------|---------|
| **备份** | 保留2-3个可替换供应商，多源供应+地理分散 | 华为囤货（6-12月安全库存） |
| **搜寻** | 行业网络、政策借力（链长制）、高管社会网络找替代 | 链长制对接平台 |
| **整合** | 自建关键环节 / 战略投资参股上游 / 联合研发 | 华为哈勃、比亚迪IGBT |
| **关系** | 长期契约（3-5年）+ 利益绑定 + 信息共享 | 旭升-特斯拉（78亿元8年订单） |

执行难点：认证周期6-18个月、技术匹配度不足、投入大周期长、过度锁定丧失弹性。

## 四、重构时间窗口

| 阶段 | 估计时间 |
|------|---------|
| 断供→危机认知 | 1-3个月 |
| 危机认知→替代搜寻 | 3-6个月 |
| 替代搜寻→供应商切换 | 6-18个月 |
| 供应商切换→稳定供应 | 12-36个月 |
| **总计** | **2-6年**（技术复杂度越高周期越长） |

华为关键节点：2019年5月断供 → 2023年8月 Mate 60 Pro（4年初步突破） → 2025年 95%国产化率（约6年系统重构完成）。一般企业周期可能更长。
