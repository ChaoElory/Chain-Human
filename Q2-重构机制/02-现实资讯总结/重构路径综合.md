## 一、重构的宏观趋势：多元化 > 本土化

### 1.1 本土化效果并不明显

以本地中间产品占总中间产品的比例衡量，2017-2023年各地区变化：

| 地区 | 本土中间品比例变化 |
|------|:---------------:|
| 墨西哥 | **+7.6 pp** |
| 东亚与太平洋 | +0.5 pp |
| 北美 | -1.4 pp |
| 欧洲与中亚 | -1.0 pp |
| 东盟 | **-3.7 pp** |

> 墨西哥本土化主因是关税规避，非政策驱动。

### 1.2 多元化是主要趋势

不同区域从其他来源进口中间产品的比例**几乎全面上升**：
- 东盟从东亚与太平洋进口中间产品 **+3.4 pp**
- 墨西哥从北美进口中间产品 **-3.5 pp**（转向其他来源）

## 二、地理格局：三大中心 + 卫星区域

```
三大中心:  北美 ──── 欧洲 ──── 东亚与太平洋
卫星区域:  墨西哥    东欧/北非  东盟
```

资本流向印证：
- 东南亚 FDI：2010年 348亿美元 → 2020年高峰 **710亿美元**
- 2018-2024年增量占2010-2024年总增量的 **57.9%**
- 来源从欧洲为主转向中国、日本、韩国为主

## 三、制造业服务化

- 研发、物流、金融、信息服务深度嵌入制造业价值链
- 东盟和墨西哥对**境外服务**依赖程度显著高于三大中心
- 境外服务业在制造业出口中的增加值贡献度持续提升

## 四、中国因素

- 中国-东盟自贸区3.0版首次提出**供应链互联互通**议题（来源：[[第4章]]）
- 中国在全球清洁能源投资中的份额从1/4上升到1/3（来源：[[第9章]]）
